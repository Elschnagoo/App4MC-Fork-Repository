h1. Roadmap

Current topics:

* Enhanced model handling to exchange different model parts
* Better mode support for modeling, like startup/shutdown phase
* Simplified hardware model
* Interactive help (visualization of model and meta model structure)
* Smaller product definition
** Remove optional parts from product downloads (can be installed via update sites)
